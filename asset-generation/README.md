# PawsMatch

A Tinder-style pet adoption web app. Swipe through real dog photos powered by The Dog API, enriched with Spanish bios generated by Gemini. Like a dog to see shelter contact details, visiting hours, and schedule a visit.

---

| Layer | Technology |

## Data Flow

```
usePets (composable)
  └─ onMounted → fillBuffer(3 cards)
       └─ petProvider.loadPet(index)
            ├─ pets[index]              ← local pets.json (name, bio, breed)
            └─ GET dog.ceo/api/.../random  ← real photo URL at response.message
                 └─ returns PetWithImage = Pet + { imageUrl }

User swipes →
  buffer.shift()          ← instant, no wait
  fillBuffer() (async)    ← refills tail in background
  decision === 'like' →
    router.push(/adoptar/:id, state: { pet })  ← AdoptionDetails reads via history.state
```

---

## Pre-fetch Buffer Strategy

`usePets` keeps 3 `PetWithImage` objects warm in memory at all times:

- **Initial mount**: 3 concurrent API calls fill the buffer before first render
- **Per swipe**: buffer head is removed instantly (zero latency), tail refill fires in background
- **`initialized` ref guard**: prevents double-execution on hot reload (Vue equivalent of `useRef` guard in `useEffect`)
- **Error handling**: per-fetch errors surface as an `error` ref, exposing a retry action in the UI; a single failure never blocks the whole deck

## Key UI Patterns

| Pattern | Implementation |
|---|---|
| Glassmorphism cards | `.glass-card` — `backdrop-blur`, `bg-white/72`, `border-white/50`, layered shadows |

## Coding Standards

### Types — `export interface` PascalCase

```ts
export interface PetWithImage extends Pet {
  imageUrl: string;  // mapped from Dog API response.message
}
```

### Services — `export const` arrow functions, async, explicit error handling

```ts
export const loadPet = async (index: number): Promise<PetWithImage | null> => {
  const res = await fetch('https://dog.ceo/api/breeds/image/random');
  if (!res.ok) throw new Error(`Dog API responded with ${res.status}`);
  const data = await res.json();
  if (data.status !== 'success') throw new Error('Dog API error');
  return { ...pets[index], imageUrl: data.message };
};
```

### Composables — `export const use*` returning reactive state

```ts
export const usePets = () => {
  const initialized = ref(false); // one-time init guard
  onMounted(async () => {
    if (initialized.value) return;
    initialized.value = true;
    await fillBuffer();
  });
  return { currentPet, swipe, matches, isLoading, error };
};
```

## Asset Generation

`asset-generation/main.py` calls Gemini to produce 50 dog profiles:

```bash
cp asset-generation/.env.example asset-generation/.env
# set GEMINI_API_KEY=your_key
cd asset-generation && uv sync && uv run main.py
```

Output shape:

```json
{ "id": 1, "name": "Luna", "bio": "...", "breed": "Galgo" }
```
